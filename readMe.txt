To run the project:
Please download the gpt4all model from link - https://huggingface.co/mrgaang/aira/blob/main/gpt4all-converted.bin

For embeddings on our model we are using - https://huggingface.co/Pi3141/alpaca-native-7B-ggml/tree/397e872bf4c83f4c642317a5bf65ce84a105786e

Please paste the models downloaded above in the models directory inside the project.

 Download the llama-cpp-python - https://github.com/abetlen/llama-cpp-python/releases
